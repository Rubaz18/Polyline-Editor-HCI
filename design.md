# Design Document

##  Goal

The goal of this system is to provide an interactive environment for:

* Drawing polylines
* Editing graphical elements
* Measuring user interaction performance

---

##  System Modes

### 1. Edit Mode

Users can:

* Create polylines
* Add points
* Delete points

### 2. Fitts' Law Mode

Users click targets and the system measures:

* Time taken
* Distance to target

---

##  HCI Principles Applied

### Affordance

Buttons and clicks indicate possible actions.

### Feedback

* Visual points and lines
* Target highlights
* Performance metrics

### Consistency

Same mouse interaction throughout system.

### Direct Manipulation

Users directly interact with objects (points).

---

##  UI Design

* Minimal interface
* Clear instructions on screen
* Color-coded elements
