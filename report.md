# HCI Project Report

##  Introduction

This project demonstrates human-computer interaction through a graphical editor and experimental tool.

---

##  Problem Statement

Design a system that:

* Allows user interaction
* Measures performance
* Applies HCI principles

---

##  Methodology

* Developed using Python and Pygame
* Implemented two modes:

  * Edit Mode
  * Experiment Mode

---

##  Implementation

* Polyline drawing system
* Point deletion logic
* Target-based experiment system

---

##  Results

* Collected time and distance data
* Displayed average performance

---

##  Conclusion

The project successfully demonstrates:

* Interactive UI design
* Application of Fitts’ Law
* Basic usability principles

---

##  Future Improvements

* Graph visualization
* Undo functionality
* Better UI design
